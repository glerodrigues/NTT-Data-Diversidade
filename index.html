O que é HTML?

HTML é uma linguagem de marcação utilizada na construção de páginas na Web. Documentos HTML podem ser interpretados por navegadores.

Titulos/Subtitulos Tag < H >

(voltar)

O HTML possui tags para destacar titulos e subtitulos em sites, essas tags são dividas em escala de 1 a 6 e são declaradas desta forma < h+NumeroNaEscala >, abaixo temos a visualizção da utilização das TAGS.

  1. Olá

  2. Olá

  3. Olá

  4. Olá

  5. Olá
  6. Olá

Notou quanto maior o numero, menor a letra? é dessa forma que o HTML trabalha!

    sub < sub >

    (voltar)

    "Subscript Element" ou subscrito é utilizado escrever um texto com letras menores e com o alinhamento abaixo das que do resto do conteúdo

    Ola meu nome é Pedro

    sup < sup >

    (voltar)

    "Superscript Element" ou sobrescrito é utilizado escrever um texto com letras menores e com o alinhamento acima das que do resto do conteúdo.

    Ola menu nome é Pedro

    blockquote < blockquote >

    (voltar)

    O Elemento HTML < blockquote> (ou Elemento HTML de citação de bloco) indica que o texto incluído é uma longa citação. Normalmente, este é processado visualmente pelo recuo. A URL para a fonte da citação pode ser dada usando o atributo citado , enquanto uma representação do texto da fonte pode ser dada usando o elemento.

    Esta é uma citação tirada da Mozilla Developer Center.

    Tag font < font >

    (voltar)

    "Font Element" ou letras é utilizado para configurar o tipo, cor e tamanho das letras de um texto de uma página HTML.

    É importante ressaltar que dentro do contexto do HTML 4.01 o marcador esta em desuso e será obsoletado no HTML 5.

    Tag del < del >

    (voltar)

    ou Elemento HTML de Texto Excluído representa uma parte do texto que foi excluído HTML (ou elemento de um documento). Este elemento é (não) renderizado pelos navegadores com uma linha entre o texto.

    Exemplo: Pedro foi riscado da lista de convidados.

    Tag abbr < abbr >

    (voltar)

    Elemento de Abreviação HTML) representa uma abreviação e opcionalmente fornece uma descrição completa para ela. Se presente, o atributo titledeve conter uma descrição completa e apenas ela.

    Exemplo: Biden é presidente dos EUA.
